# NekoStream CLI 

NekoStream CLI là một ứng dụng dòng lệnh xem anime tốc độ cao, siêu nhẹ, tích hợp trình phát video đa tính năng và hỗ trợ lấy dữ liệu từ nhiều nguồn (providers) phổ biến tại Việt Nam.

##  Cài đặt & Sử dụng

Yêu cầu: [Node.js](https://nodejs.org/) (phiên bản 18+).

Mở terminal và chạy lệnh cài đặt toàn cục:
```bash
npm install -g nekostream
```

Sau khi cài đặt xong, bạn có thể gọi ứng dụng từ bất kỳ đâu:
```bash
nekostream
```


## Tính năng chính
- Tốc độ cao, gọn nhẹ, điều khiển hoàn toàn bằng phím tắt.  
- Theo dõi lịch sử xem phim và tính năng "Tiếp tục xem".
- Thay đổi tùy chọn Domain của Provider linh hoạt.
- Trình phát (Player) tự động phát tập tiếp theo.

## Tuyên bố miễn trừ trách nhiệm
- **Phi lợi nhuận**: Dự án không nhằm mục đích thương mại hóa, được tạo ra hoàn toàn vì cộng đồng.
- **Không lưu trữ**: NekoStream CLI đóng vai trò là cầu nối (client). Chúng tôi không lưu trữ, không phát tán bất cứ thông tin hay nội dung số nào ngoài mã nguồn trên thư mục Github này.
- **Tính riêng tư**: Dự án KHÔNG thu thập thông tin của bạn, không sử dụng bất kỳ máy chủ trung gian nào. Mọi request (kết nối) đều là kết nối trực tiếp từ máy tính của bạn đến máy chủ Provider.
- **Mã nguồn mở**: Dự án có sử dụng AI trong quá trình phát triển. Bạn có thể thoải mái clone và tham khảo code, nhưng vui lòng để lại credit. Cảm ơn!
